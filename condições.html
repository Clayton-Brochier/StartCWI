Condições em Javascript

Antes de iniciar falar de condições, vamos falar de Sequências:
Exemplo:
1º var n = 3, aqui atribuimos o valor de 3 a variável n
2º n += 2, agora "n" que vale 3 é somado a "2" e "n" passa valer 5.
3º window.alert(n)

Aqui é executado uma sequência e não pode ser alterado a ordem.

Agora vamos falar sobre as Condições:
Com as condições, o programa não é executado em uma linha vertical, como no exemplo acima.
Agora em determinado ponto, deve ser tomado uma decisão. Em determinado ponto,
há uma bifurcação, que você deve informar a condição que o programa irá seguir.
Se for verdadeira, ele segue a primeira opção, se for falsa, ele segue para a segunda opção.
Exemplo:
if(condição){
//Código a ser executado caso a condição seja verdadeira
} else {
//Código a ser executado caso a condição seja falsa
}

Tipos de condições:

1° - Condições Simples (comparação):
if(condição){
//condição seja verdadeira
}
2° - if(condição){
//verdadeira
} else {
//falsa
}
